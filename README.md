# JWT Authentication using Prisma and Express 

Endpoints:

* register
  * localhost:5000/api/v1/auth/register
* login
  * localhost:5000/api/v1/auth/login
* refresh_token
  * localhost:5000/api/v1/auth/refresh_token
* get profile
  * localhost:5000/api/v1/user/profile

## Setup

```
npm install
```

## Lint

```
npm run lint
```

## Test

```
npm run test
```

## Development

```
npm run dev
```
